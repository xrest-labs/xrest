# Features

xrest is packed with features designed for power users and microservice developers.

## [Service Client](./services)
Learn how xrest organizes requests into Services and how to manage them.

## [Git Integration](./git)
xrest is built to work with Git. Learn how to sync your API configurations with your team.

## [Environments](./environments)
Manage complex environment variable sets for Dev, Stage, and Production.

## [Safe vs Unsafe](./safe-unsafe)
Understand the safety features that prevent accidental production disasters.

## [Pre-flight Requests](./preflight-requests)
Automate your authentication flows with intelligent pre-flight token management.

## [History](./history)
Keep track of past executions and quickly re-run previous requests.
